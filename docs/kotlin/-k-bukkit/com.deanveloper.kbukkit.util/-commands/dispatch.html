dispatch

@JvmStatic fun dispatch(sender: CommandSender, cmdLine: String): Boolean

Dispatches a command on this server, and executes it if found.

Parameters

sender - the apparent sender of the command

cmdLine - the command + arguments. Example: test abc 123

Exceptions

CommandException - thrown when the executor for the given command fails with an unhandled exception

Return
false if no target is found
